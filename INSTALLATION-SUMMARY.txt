PROJECT OVERVIEW
================================================================================
This project provides comprehensive torrent and P2P traffic blocking for 
servers and VPS. It prevents your server from being blocked by datacenters 
due to illegal torrent usage.

USEFUL COMMANDS (Enhanced)
================================================================================

View blocking statistics:
$ cat /var/log/torrent-blocking/blocking-stats.txt

View blocked attempts:
$ tail -f /var/log/torrent-blocking/blocked-domains.log

Check update history:
$ cat /etc/torrent-blocklists/update.log

Update blocklists manually:
$ sudo /opt/torrent-blocking/update-blocklist.sh

Monitor blocking:
$ /opt/torrent-blocking/monitor-blocking.sh

Check active rules (iptables):
$ sudo iptables -L -n | grep DROP

Check active rules (nftables):
$ sudo nft list ruleset | grep drop

Test domain blocking:
$ nslookup 1337x.com
(Should return 0.0.0.0 or timeout)

TROUBLESHOOTING
================================================================================

Installation fails:
1. Verify sudo access: sudo whoami
2. Check internet: ping 8.8.8.8
3. Check wget: which wget

Blocking not working:
1. Verify firewall running: sudo systemctl status iptables
2. Check rules: sudo iptables -L -n | head -20
3. Test DNS: nslookup 1337x.com

High CPU usage:
1. Use nftables instead of iptables
2. Reduce rule count with DNS blocking
3. Check logs for excessive blocks
